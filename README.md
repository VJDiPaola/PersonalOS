# PersonalOS

A markdown-first personal operating system starter for builders who want a private, version-controlled dashboard for projects, tasks, decisions, reviews, and AI-ready context.

## What This Repo Is

This repo is the public starter/template version of a private personal operating system.

It is meant to become:

- a clean starter for your own operating system
- a reference architecture for markdown-first personal dashboards
- a schema-first foundation for generated views, weekly reviews, and AI context exports

## Current Status

This repo currently starts with the V2 blueprint and will grow into a usable starter template.

Start here:

- [Blueprint](docs/blueprint.md)

## Direction

The intended architecture is:

- canonical schemas
- stable IDs for entities
- a single CLI
- generated human dashboards
- generated AI exports
- a clean split between source content and generated views

## Planned Structure

```text
schema/
entities/
views/
exports/
cli/
tests/
docs/
```

## Why A Separate Public Repo

The real operating system stays private so it can remain useful and personal.

This public repo exists so the architecture, tooling, and starter experience can be shared without exposing private notes, contacts, applications, or strategy.
